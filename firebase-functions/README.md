# Firebase Functions (JavaScript)

## Setup

1. Install dependencies:
   - `npm install`

2. Configure environment variables via Firebase:
   - `firebase functions:config:set azure.doc_intel_endpoint="..." azure.doc_intel_key="..."`
   - `firebase functions:config:set openai.api_key="..."`

3. Run emulator:
   - `npm run serve`

4. Deploy:
   - `npm run deploy`
